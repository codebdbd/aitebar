# Руководство пользователя AiteBar

## 1. Что такое AiteBar

`AiteBar` — это скрываемая боковая панель быстрого доступа для Windows. Она появляется у выбранного края экрана и позволяет:

- запускать сайты, программы, файлы, папки, команды и скрипты;
- хранить кнопки в нескольких панелях;
- быстро открывать встроенные инструменты;
- вызывать панель мышью, через tray или горячими клавишами.

## 2. Установка и первый запуск

1. Установите программу через `AiteBar-Setup.exe`.
2. Запустите `AiteBar`.
3. После запуска программа сворачивается в tray и начинает работать в фоне.
4. Если приложение уже запущено, второй экземпляр не стартует.

## 3. Основные элементы интерфейса

В программе есть три главные точки взаимодействия:

- рабочая панель на краю экрана;
- меню в системном tray;
- окна настройки кнопок и общих параметров программы.

## 4. Как открыть панель

Панель можно открыть несколькими способами:

- подвести курсор к активной зоне у выбранного края экрана;
- щелкнуть по иконке программы в tray левой кнопкой мыши;
- выбрать `Открыть` в контекстном меню tray;
- использовать назначенную горячую клавишу.

Панель автоматически скрывается, когда вы кликаете вне ее области.

## 5. Как перемещать рабочую панель

Основной способ смены положения панели — перетаскивание самой рабочей панели.

Как это работает:

1. Найдите небольшое `ушко`/handle на краю панели.
2. Захватите его мышью.
3. Перетащите панель к другому краю экрана.
4. Отпустите кнопку мыши.

После этого программа сохраняет:

- новую сторону панели;
- монитор, на который вы перетащили панель.

Если мыши нет или вы предпочитаете настраивать положение вручную, это также можно сделать в `Настройках программы`.

## 6. Структура панели

Рабочая панель состоит из двух частей:

- служебная область:
  - кнопка добавления;
  - встроенные быстрые инструменты;
- пользовательские кнопки активной панели.

Пользовательские кнопки можно:

- запускать обычным кликом;
- перетаскивать для изменения порядка;
- редактировать, перемещать и удалять через контекстное меню.

## 7. Панели

В `AiteBar` есть четыре панели кнопок.

По умолчанию они называются:

- `Панель 1`
- `Панель 2`
- `Панель 3`
- `Панель 4`

Вы можете переименовать их под свои задачи, например:

- `Работа`
- `ИИ`
- `Личное`
- `Медиа`

Если имя панели изменено, программа везде показывает именно пользовательское имя.

Примеры:

- в выборе панели при создании кнопки;
- в горячих клавишах;
- в пункте `Переместить в: ...`.

## 8. Быстрый старт

Рекомендуемый первый сценарий:

1. Откройте панель.
2. Нажмите кнопку добавления `+`.
3. Укажите:
   - название кнопки;
   - тип действия;
   - панель;
   - иконку и цвет.
4. Сохраните кнопку.
5. При необходимости откройте `Настройки программы` и переименуйте панели.

## 9. Создание и редактирование кнопки

Окно `Настройки` используется для добавления и редактирования кнопки.

Основные поля:

- `Название кнопки`
- `Тип действия`
- `Панель`
- значение действия:
  - URL;
  - путь к программе;
  - путь к файлу;
  - путь к папке;
  - путь к скрипту;
  - команда;
  - сочетание клавиш
- иконка;
- цвет иконки.

### Доступные типы действий

- `Ссылка`
- `Программа`
- `Файл`
- `Папка`
- `Скрипт`
- `Команда`
- `Горячие клавиши`

### Поддерживаемые действия по типам

#### Ссылка

Используется для открытия сайта в браузере.

Доступны параметры:

- `Браузер`
- `Профиль`
- `В режиме приложений`
- `В режиме инкогнито`
- `Открыть на весь экран`
- `В режиме ротации`

В текущем интерфейсе доступны браузеры:

- `Chrome`
- `Edge`
- `Brave`
- `Yandex`
- `Firefox`

#### Программа

Запускает `.exe`, `.lnk` и другие подходящие исполняемые файлы.

#### Файл

Открывает выбранный файл в системе или в ассоциированном приложении.

#### Папка

Открывает выбранную папку.

#### Скрипт

Запускает файл скрипта, например:

- `.bat`
- `.cmd`
- `.ps1`
- `.py`

#### Команда

Выполняет консольную команду.

#### Горячие клавиши

Отправляет настроенное сочетание клавиш.

## 10. Иконки и цвета кнопок

Для кнопки можно выбрать:

- иконку из встроенной библиотеки;
- собственную иконку;
- цвет из палитры;
- цвет вручную через `HEX код`.

Если выбрана пользовательская картинка, она сохраняется в каталоге данных программы.

## 11. Контекстное меню кнопки

Щелкните по пользовательской кнопке правой кнопкой мыши.

Доступные действия:

- `Редактировать`
- `Переместить в: <имя панели>`
- `Удалить`

Особенности:

- при удалении кнопки панель не должна закрываться;
- кнопку можно удалить и сразу продолжить работу с открытой панелью;
- при переносе кнопка перемещается в выбранную панель без ручного копирования.

## 12. Изменение порядка кнопок

Пользовательские кнопки можно переставлять внутри текущей панели.

Как это сделать:

1. Зажмите кнопку левой кнопкой мыши.
2. Потяните ее вдоль панели.
3. Отпустите в нужной позиции.

После этого порядок сохраняется.

## 13. Встроенные быстрые инструменты

В программе есть встроенные быстрые инструменты:

- `Поиск`
- `Скриншот`
- `Видео`
- `Калькулятор`

Их отображение включается и выключается в `Настройках программы`.

## 14. Импорт и экспорт панелей

В программе можно переносить содержимое панели между компьютерами или проектами.

Для этого используются два действия:

- `Импорт в текущую панель...`
- `Экспорт текущей панели...`

Они доступны:

- в контекстном меню самой панели;
- в меню программы в tray.

### 14.1. Экспорт текущей панели

Экспорт сохраняет активную панель в один файл с расширением `.aitebarpanel`.

В пакет входят:

- кнопки текущей панели;
- порядок кнопок;
- параметры действий;
- встроенные иконки и цвета;
- пользовательские файловые иконки, если они есть.

Имя экспортируемой панели записывается в пакет как metadata.

### 14.2. Импорт в текущую панель

Импорт добавляет кнопки из файла `.aitebarpanel` в активную панель, которая выбрана в данный момент.

Особенности импорта:

- импорт всегда идет в текущую панель;
- имя текущей панели не меняется;
- импортированные кнопки добавляются в конец списка;
- если в пакете есть файловые иконки, программа пересохраняет их в локальную папку данных;
- если файл иконки отсутствует или поврежден, кнопка все равно импортируется с резервной встроенной иконкой.

Перед импортом программа показывает подтверждение с количеством кнопок.

В первой версии отмена уже выполненного импорта не поддерживается.

## 15. Настройки программы

Окно `Настройки программы` содержит четыре блока.

### 15.1. Общие параметры

Здесь настраиваются:

- `Позиция панели`
- `Монитор`
- `Размер панели`
- `Зона активации, %`
- `Задержка появления, мс`

#### Позиция панели

Доступные варианты:

- `Верх`
- `Право`
- `Лево`
- `Низ`

#### Монитор

Позволяет выбрать экран, на котором работает панель.

#### Размер панели

Определяет размер панели вдоль активной оси.

#### Зона активации, %

Определяет размер области, которая реагирует на подведение мыши.

#### Задержка появления, мс

Определяет задержку перед показом панели после входа курсора в зону активации.

### 15.2. Имена панелей

Здесь можно задать имена для четырех панелей.

Поля:

- `1`
- `2`
- `3`
- `4`

Если оставить имя пустым, используется стандартное имя:

- `Панель 1`
- `Панель 2`
- `Панель 3`
- `Панель 4`

### 15.3. Горячие клавиши

Здесь настраиваются:

- `Показать панель`
- `Общий модификатор`
- `Следующая панель`
- `Предыдущая панель`
- `Панель 1`
- `Панель 2`
- `Панель 3`
- `Панель 4`

Смысл настроек:

- `Показать панель` — открывает панель глобальной горячей клавишей;
- `Общий модификатор` — общий модификатор для действий переключения и перехода между панелями;
- `Следующая панель` и `Предыдущая панель` — переключают панели;
- `Панель 1..4` — прямой переход к конкретной панели.

Если у вас уже были разные модификаторы для разных панелей, окно настроек должно сохранить их корректно и не переписать молча.

### 15.4. Быстрые инструменты

Здесь включаются и выключаются:

- `Поиск`
- `Скриншот`
- `Видео`
- `Калькулятор`

## 16. Меню tray

Контекстное меню иконки программы в системном tray содержит:

- `Открыть`
- `Настройки программы`
- `О программе`
- `Справка`
- `Поддержать автора`
- `Закрыть и выйти`

Левый клик по иконке tray также открывает панель.

## 17. Где хранятся настройки и данные

Данные программы хранятся в профиле пользователя Windows:

`%AppData%\Codebdbd\Aite Bar`

Основные файлы:

- `custom_buttons.json` — пользовательские кнопки;
- `settings.json` — общие настройки программы;
- `error.log` — журнал ошибок.

Пользовательские иконки сохраняются в папке:

`%AppData%\Codebdbd\Aite Bar\Icons`

## 18. Типичные сценарии использования

### Сценарий 1. Рабочая панель

Пример:

- `Панель 1` = рабочие сайты и инструменты
- браузер в `режиме приложений`
- быстрый переход через горячую клавишу

### Сценарий 2. Панель ИИ

Пример:

- `Панель 2` = ChatGPT, Claude, Gemini, локальные инструменты
- разные профили браузера;
- быстрый запуск через пользовательские кнопки.

### Сценарий 3. Личное и повседневное

Пример:

- `Панель 3` = почта, заметки, файлы, папки, калькулятор

### Сценарий 4. Системные и служебные задачи

Пример:

- `Панель 4` = скрипты, команды, служебные ярлыки

## 19. Советы по настройке

- Не пытайтесь уместить все кнопки в одну панель: разделите задачи по четырем панелям.
- Давайте панелям короткие и понятные имена.
- Для сайтов, которые используются как отдельные инструменты, попробуйте `В режиме приложений`.
- Для полноэкранных сценариев используйте `Открыть на весь экран`.
- Для кнопок со скриптами и командами проверяйте путь и рабочее окружение заранее.

## 20. Устранение неполадок

### Панель не появляется

Проверьте:

- выбран ли правильный `Монитор`;
- не слишком ли маленькая `Зона активации, %`;
- не слишком ли большая `Задержка появления, мс`;
- не перекрывается ли край экрана другими окнами или интерфейсом.

### Кнопка не запускается

Проверьте:

- корректность пути или URL;
- доступность файла или папки;
- существование выбранного браузера;
- права доступа к скрипту или команде.

### Не видно нужной панели

Проверьте:

- имя панели в `Настройках программы`;
- назначенные горячие клавиши;
- содержимое активной панели.

### Программа ведет себя нестабильно

Проверьте файл:

`%AppData%\Codebdbd\Aite Bar\error.log`

## 21. Краткая памятка

- Открыть панель: край экрана, tray или горячая клавиша.
- Переместить панель: перетащить handle на самой панели.
- Добавить кнопку: нажать `+`.
- Изменить кнопку: правый клик -> `Редактировать`.
- Перенести кнопку: правый клик -> `Переместить в: ...`.
- Удалить кнопку: правый клик -> `Удалить`.
- Настроить программу: tray -> `Настройки программы`.
