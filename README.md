# AiteBar

AiteBar — это удобная скрытая панель быстрого доступа для Windows. Она появляется при наведении курсора на край экрана и предоставляет быстрый доступ к вашим любимым нейросетям, сайтам, утилитам, скриптам и системным командам.

## Особенности
- **Скрытая панель**: Отображается только при поднесении курсора к левому краю экрана.
- **Поддержка профилей Chrome**: Запускайте сайты в нужном профиле, в режиме "Инкогнито" или "App Mode".
- **Ротация профилей**: Автоматическое переключение профилей Chrome при каждом запуске кнопки.
- **Горячие клавиши**: Назначайте глобальные комбинации (Ctrl, Shift, Alt, Win + клавиша) для вызова действий.
- **Импорт и экспорт панелей**: Сохраняйте текущую панель в один файл `.aitebarpanel` и импортируйте готовые наборы кнопок в активную панель.
- **Разнообразие действий**:
  - Открытие URL прямо в Chrome
  - Запуск программ и ярлыков (.exe, .lnk)
  - Запуск скриптов (.bat, .cmd, .ps1, .py)
  - Выполнение консольных команд (через cmd.exe с подтверждением)
- **Богатая библиотека иконок**: Встроенные кроссплатформенные Material Symbols, Fluent System Icons и Font Awesome Brands. Настройки цвета и точное позиционирование.

## Установка и запуск
1. Загрузите релиз из соответствующего раздела.
2. Запустите `AiteBar-Setup.exe`.
3. Установите программу как обычное Windows-приложение.
4. После установки запустите `AiteBar` из меню Пуск. Программа появится в системном трее Windows.

## Управление
- **Настройка кнопок**: Кликните ПКМ (правой кнопкой мыши) по любой кнопке для её редактирования.
- **Добавление новой**: Нажмите кнопку `+` в конце любого блока меню.
- **Удаление**: Очистите название кнопки, либо установите пустые параметры, и она удалится.
- **Импорт/экспорт панели**: Используйте контекстное меню панели или меню в tray для импорта в текущую панель и экспорта текущей панели.

## Требования
- ОС: Windows 10 / Windows 11

## Хранение данных
- Конфигурация и пользовательские данные хранятся в `C:\Users\user\AppData\Roaming\Codebdbd\Aite Bar`
